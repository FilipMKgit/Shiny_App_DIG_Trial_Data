# Shiny_App_DIG_Trial_Data
This Repository Contains A shiny application for exploring the data In the DIG trial. 

To run inside RStudio enter command:
runGitHub("Shiny_App_DIG_Trial_Data", "FilipMKgit")


To run this app **directly from the cloned repository**:

1. Clone the repo:  

2. In the Repository URL box paste: https://github.com/FilipMKgit/Shiny_App_DIG_Trial_Data

3. Open the project in R or RStudio.

4. Install required packages (if needed):

5. Use shiny::runApp()

To oepn the app online use:

 https://tomfilipdigapp.shinyapps.io/shiny_app_dig_trial_data/
